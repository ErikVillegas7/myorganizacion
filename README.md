# Mi Organización

Plataforma para organizar notas rápidas, calendario académico, materias con tablero y seguimiento de hábitos. Los datos se guardan en el navegador usando localStorage.

## Vistas principales

- `/notas`: jerarquía de carpetas con notas editables.
- `/calendario`: eventos académicos con temporizador de próximos hitos.
- `/materias`: tablero por unidades, notas y condiciones de cursada.
- `/habitos`: seguimiento semanal con diagrama de progreso.

## Desarrollo

```bash
npm run dev
```

## Build

```bash
npm run build
```
