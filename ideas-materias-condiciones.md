# Idea futura: condiciones de materias

Agregar en su momento una mejora para manejar las condiciones de las materias, por ejemplo:

- Cuestionarios
- Trabajos grupales
- Entregas parciales
- Presentaciones orales
- Proyectos finales
- Actividades prácticas

Esto permitiría que cada materia tenga un seguimiento más completo de los requisitos y evaluaciones, en lugar de solo usar una etiqueta genérica de condición.

Además, cada vista debería tener un botón de información o un icono `!` para mostrar una breve guía del espacio: "en esta página podés...". Así el usuario entiende mejor qué puede hacer en cada pantalla.
